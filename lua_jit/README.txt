This is lua jit (just in time compiler). Use it to speed up your lua code.

1.0 # INSTALLATION : 

    git clone https://luajit.org/git/luajit.git             **download
	cd luajit/ ; make && sudo make install                  **build
	sudo ln -sf luajit-2.1.0-beta3 /usr/local/bin/luajit    ** add to path

2.0 # RUN LUA USING LUAJIT 

    luajit your_script.lua
